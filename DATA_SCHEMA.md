# 河憩寮網站｜Google Sheets 資料欄位規格

網站資料主要由 Google Sheets 發布成 CSV 後讀取。發布網址必須是 CSV 格式。

正確格式：`https://docs.google.com/spreadsheets/d/e/.../pub?gid=工作表ID&single=true&output=csv`

錯誤格式：`https://docs.google.com/spreadsheets/d/e/.../pubhtml?gid=工作表ID&single=true`

## 1. 本週名單／活體商品

建議欄位：`id, 品名, 學名, 分類, 價格, 尺寸, 狀態, 標籤, 餵食, 照片網址, 備註, 是否售完, 庫存/數量`

活體狀態規則：

- `是否售完 = 是` 時，應優先視為不可販售。
- 狀態含「暫不販售、不可販售、停售、保留、休養」時，應關閉詢問按鈕或標示不可安排。
- 「可詢問」不可解釋成保證可出貨。

## 2. 造景介紹

建議欄位：`排序, 分類, 類型名稱/造景名稱/名稱, 介紹文字, 適合魚種/適合方向, 尺寸/參考尺寸, 注意事項/選購重點, 備註, 圖片網址, 狀態`

注意：

- 「適合魚種」只能作方向參考，不可寫成絕對適合。
- 需提醒實際仍依缸體尺寸、水流、底砂、遮蔽、個體性格與混養狀況評估。

## 3. 設備介紹

建議欄位：`排序, 分類, 設備名稱/名稱, 介紹文字/用途/說明, 適合情境/適用情境, 注意事項/選購重點, 備註, 圖片網址, 狀態`

設備介紹是教學區，不是販售區。

## 4. 硬體販售

建議欄位：

| 欄位 | 說明 |
|---|---|
| 商品ID | 同一商品不同規格需填相同商品ID |
| 排序 | 規格排序／商品排序 |
| 分類 | 例如過濾設備、打氣設備、水族工具、水質處理、造景底砂 |
| 品名 | 商品名稱 |
| 商品說明 | 商品主要說明，可用 `・` 條列 |
| 規格 | 例如 S、M、Tidal 35、250ml |
| 狀態 | 現貨、少量、補貨中、暫缺 |
| 售價 | 單一規格售價 |
| 規格備註 | 會接在規格名稱後方，例如適用水量、尺寸、用途 |
| 圖片網址 | 該規格圖片，可與其他規格不同 |
| 商品狀態 | 顯示／隱藏，可選 |

### 硬體多規格合併邏輯

同一個「商品ID」會合併成同一張商品卡。

範例：

| 商品ID | 分類 | 品名 | 規格 | 售價 |
|---|---|---|---|---|
| tidal-filter-01 | 過濾設備 | Seachem Tidal 外掛過濾器 | Tidal 35 | 1960 |
| tidal-filter-01 | 過濾設備 | Seachem Tidal 外掛過濾器 | Tidal 55 | 2240 |
| tidal-filter-01 | 過濾設備 | Seachem Tidal 外掛過濾器 | Tidal 75 | 2800 |

前台應顯示為一張商品卡，底下列出三個規格。

### 規格備註顯示方式

規格備註應直接接在規格名稱後方：

`Tidal 35｜50L 以下魚缸｜尺寸約 19.5 × 15.5 × 11.5 cm`

不要再另外做很長的「規格補充」區塊，避免卡片過長。

### 商品說明建議格式

使用 `・` 條列，前台會整理成較乾淨的說明：

```text
・使用前請先搖勻
・一般建議用量：每 250L 水量使用 5ml
・第一次使用建議先以半量添加，觀察魚蝦、水草與藻況反應後再逐步調整
```

## 5. 常見硬體分類建議

| 分類 | 適合商品 |
|---|---|
| 過濾設備 | 水妖精、外掛、內置、圓筒、上部過濾、替換棉、濾材 |
| 打氣設備 | 空氣馬達、風管、逆止閥、調節閥、氣泡石、分接頭 |
| 水族工具 | 滴管、吸管、餵食管、量杯、夾子、刮刀、吸便器、換水管 |
| 水質處理 | 除氯劑、水質穩定劑、肥料、添加劑 |
| 加溫設備 | 加溫棒、控溫器、溫度計 |
| 照明設備 | LED 燈、水草燈、觀賞燈 |
| 降溫設備 | 水族風扇、冷水機 |
| 造景素材 | 底砂、石材、沉木、枯葉、背景素材 |
